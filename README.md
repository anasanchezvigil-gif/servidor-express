# API Proyecto - Servidor Express Estudiantes 🇸🇻

Este es un proyecto backend desarrollado con Node.js y Express para gestionar un listado de estudiantes (CRUD en memoria).

## Componentes del Proyecto

* **index.js**: Servidor Express con el arreglo de estudiantes y 5 endpoints configurados.
* **package.json**: Configuración de dependencias del proyecto.
* **.gitignore**: Configuración para excluir carpetas innecesarias como `node_modules`.
